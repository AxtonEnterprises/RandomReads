# Random Reads PWA

A mobile-first React + Vite PWA starter for Random Reads / The Literature Foundation.

## Features

- React + Vite
- PWA install support
- Mobile-first layout
- Random book discovery
- Public domain book search through Gutendex
- Reader page
- Local saved books
- Local reading notes

## Run locally

```bash
npm install
npm run dev
```

Open the local network URL on your phone while your computer and phone are on the same Wi-Fi.

## Build

```bash
npm run build
npm run preview
```
